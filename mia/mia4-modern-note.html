Slide — Per-Example Calibration

Analogy dropped from slide for density: per-example calibration is like per-question grading on an exam. A globally hard question (e.g., a memorized rare example) needs its own threshold; a globally easy one needs another. One cutoff for all questions (Yeom's global threshold) loses information about point difficulty.

Slides 147 + 147b — MIA Pipeline Comparison (full pseudocode)

The slide deck splits this comparison across two slides for readability. The full side-by-side pseudocode is reproduced here for after-talk reference.

Threshold attack — simplest baseline

def threshold_attack(model, z, τ):
    return loss(model, z) < τ

One scalar threshold, no shadow models, no calibration. A point with low loss is declared a member. Strong on average accuracy; very weak at low FPR because it cannot account for per-example difficulty.

LiRA — most powerful (Carlini et al. 2022)

def lira_attack(target, z, shadows):
    ℓ_in  = [loss(s, z) for s in shadows if z ∈ s.train]
    ℓ_out = [loss(s, z) for s in shadows if z ∉ s.train]
    μ_in,  σ_in  = gaussian_fit(ℓ_in)
    μ_out, σ_out = gaussian_fit(ℓ_out)
    return log_likelihood_ratio(loss(target, z), μ_in, σ_in, μ_out, σ_out)

For each target point $z$, fit a Gaussian over "if-in" shadow losses and another over "if-out" shadow losses, then compute the per-example log-likelihood ratio. Implements the Neyman–Pearson optimal test approximately (under Gaussian loss). Cost: $\mathcal{O}(N)$ shadow models per attack budget; typical $N \approx 256$.

RMIA — best cost/accuracy tradeoff (Zarifzadeh et al. 2024)

def rmia_attack(target, z, ref_models, ref_data):
    ratio_z  = mean([loss(target, z) / loss(r, z) for r in ref_models])
    ratio_zp = [mean([loss(target, z') / loss(r, z') ...]) for z' in ref_data]
    return percentile_rank(ratio_z, ratio_zp)

Replace the per-example "in vs. out" Gaussian fit with a population-relative ratio. For each candidate $z$, compare the target/reference loss ratio at $z$ to the same ratio at a population sample. The percentile rank is the score. Achieves near-LiRA TPR@low-FPR with only $2$–$8$ reference models — a $30$–$100\times$ cost reduction.

Reading the comparison

  • Threshold answers: "is the loss small in absolute terms?" — ignores point difficulty.
  • LiRA answers: "is the loss small relative to what we'd expect if $z$ were not in the training set?" — calibrates per-example, expensive.
  • RMIA answers: "is the target/reference loss ratio at $z$ unusually large compared to the rest of the population?" — calibrates against the population instead of the per-example shadow distribution, cheap.
