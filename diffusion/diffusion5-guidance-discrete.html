01

Classifier Guidance

Bayes decomposition and the $\omega$-scaled reverse SDE

Conditional generation

So far we have sampled from $P_{\mathbf{X}}$. In practice we want $P_{\mathbf{X}\,|\,Y}$ for a target label or prompt $Y$ (ImageNet classes, text prompts, segmentation masks, $\ldots$).

Naive approach. Train a separate conditional score $s_\theta(\mathbf{x}\,|\,Y=y)$ for each value of $y$. This is expensive and does not scale beyond a handful of labels.

Classifier guidance (Dhariwal & Nichol, 2021): reuse the pretrained unconditional score and adjust sampling with a small auxiliary classifier.

Bayes decomposition of the conditional score

From Bayes' rule,

$$P_{\mathbf{X}\,|\,Y}(\mathbf{x}\,|\,y) = \frac{P_{\mathbf{X}}(\mathbf{x})\,P_{Y\,|\,\mathbf{X}}(y\,|\,\mathbf{x})}{P_Y(y)}.$$

Taking $\nabla_{\mathbf{x}} \log$ of both sides:

$$\nabla_{\mathbf{x}} \log P_{\mathbf{X}\,|\,Y}(\mathbf{x}\,|\,y) = \nabla_{\mathbf{x}} \log P_{\mathbf{X}}(\mathbf{x}) + \nabla_{\mathbf{x}} \log P_{Y\,|\,\mathbf{X}}(y\,|\,\mathbf{x}).$$

The $P_Y(y)$ factor has no $\mathbf{x}$ dependence, so it drops. The conditional score is the unconditional score plus the gradient of a classifier.

A classifier on noisy inputs

At sampling time we operate on the noisy state $\mathbf{X}^{(n)}$, so we need a classifier that also works at that noise level:

$$c_\phi(\mathbf{X}^{(n)}, y, n) \;\approx\; P_{\mathbf{X}^{(n)}}(Y = y\,|\,\mathbf{X}^{(n)}).$$

This is a regular classifier taking the noise index $n$ as an extra input. We train it on noisy images generated by the forward process.

Intuition: in early generation (large $n$) the classifier only needs a coarse guess. Only near the end (small $n$) does it need to be accurate. Getting $Y^{(n)} \approx y$ for small $n$ is enough.

Classifier-guided reverse SDE

Replace the unconditional score in Anderson's reverse SDE by the Bayes-decomposed conditional:

$$d\bar{\mathbf{X}}_t = \Bigl(f - g^2\bigl(\nabla\log p_t(\bar{\mathbf{X}}_t) + \nabla \log c_\phi(\bar{\mathbf{X}}_t, y, t)\bigr)\Bigr)dt + g\,d\bar{\mathbf{Z}}_t.$$

In practice, amplifying the classifier term by a factor $\omega > 1$ produces sharper, more label-consistent samples at the cost of some diversity:

$$d\bar{\mathbf{X}}_t = \Bigl(f - g^2\bigl(\nabla\log p_t(\bar{\mathbf{X}}_t) + \omega\,\nabla \log c_\phi(\bar{\mathbf{X}}_t, y, t)\bigr)\Bigr)dt + g\,d\bar{\mathbf{Z}}_t.$$

Empirically, $\omega$ trades coverage of the class distribution against class fidelity. Larger $\omega$ collapses onto class modes.

02

Inpainting

Conditioning on a fixed subset of the signal

Setup

Let $\Omega$ be a pixel-subsampling operator: $Y = \Omega(X^{(0)})$ is the observed part of the image, and $\bar{\Omega}$ returns the complementary part to be generated.

Let $Z^{(t)} = \bar{\Omega}(X^{(t)})$. It follows its own SDE on the lower-dimensional complement:

$$dZ^{(t)} = \bar{f}\bigl(Z^{(t)}, t\bigr)\,dt + \bar{g}(t)\,d\bar{W}^{(t)},\qquad \bar{f} = \bar{\Omega}(f),\;\bar{g} = \bar{\Omega}(g),\;\bar{W} = \bar{\Omega}(W).$$

The Brownian motion decomposes cleanly because it is coordinate-wise independent.

Approximating the conditional score

We want $p_t\bigl(Z^{(t)}\,|\,\Omega(X^{(0)}) = Y\bigr)$. Condition on the noisy version of the observation and marginalize:

$$\begin{aligned} &p_t\bigl(Z^{(t)}\,|\,\Omega(X^{(0)}) = Y\bigr) \\ &\quad\approx \mathbb{E}_{\Omega(X^{(t)})\,\sim\,p_t(\cdot\,|\,\Omega(X^{(0)}))}\!\bigl[p_t(Z^{(t)}\,|\,\Omega(X^{(t)}))\bigr] \\ &\quad\approx p_t\bigl(Z^{(t)}\,|\,\hat{\Omega}(X^{(t)})\bigr), \end{aligned}$$

where $\hat{\Omega}(X^{(t)})$ is a noise-corrupted version of $Y$ at time $t$, generated by running the forward process on the observed pixels alone. The second line is a single-sample approximation of the expectation.

Inpainting sampler

Score-Guided Inpainting
  1. Sample $X^{(N)} \sim \mathcal{N}(0, I)$ on the full image.
  2. For $n = N, N{-}1, \ldots, 1$:
    1. Sample $\epsilon_n \sim \mathcal{N}(0, I)$, form $\zeta_n = \beta_n\,Y + \gamma_n\,\epsilon_n$ (noised observation at level $n$).
    2. Run one reverse-SDE step using $\bar{\Omega}\bigl(s_\theta([\bar{Z}^{(n)}; \zeta_n], n)\bigr)$ as the guidance on the generated pixels.

Each step pairs the hidden pixels being generated with a freshly noised copy of the known pixels, so the score network always sees a consistent full image.

03

Classifier-Free Guidance

Guidance without an extra classifier

Why drop the classifier?

Classifier guidance requires a second network, trained on noisy inputs, whose gradients have to be evaluated at every sampling step. Two model copies at inference is expensive, and the classifier can be hard to train well in the high-noise regime.

An equivalent rewrite of the guided score:

$$\nabla \log p_t(\mathbf{x}) + \omega\,\nabla \log p_t(y\,|\,\mathbf{x}) \;\equiv\; \omega\,\nabla \log p_t(\mathbf{x}\,|\,y) + (1-\omega)\,\nabla \log p_t(\mathbf{x}).$$

If we have both the conditional and unconditional scores, we never need a classifier. Classifier-free guidance (Ho & Salimans, 2022) trains a single model that provides both.

A dual-role network

Train $\epsilon_\theta(\mathbf{X}^{(n)}, y, n)$ where $y$ takes the label value or the null token $\varnothing$:

$$\epsilon_\theta(\mathbf{X}^{(n)}, y, n) \;\approx\; \text{conditional noise under label } y,$$
$$\epsilon_\theta(\mathbf{X}^{(n)}, \varnothing, n) \;\approx\; \text{unconditional noise.}$$

Training trick: during training, replace $y$ by $\varnothing$ with probability $p_\text{drop}$ (typically $0.1$–$0.2$). The network learns both conditionals in a single pass.

The per-step loss is unchanged from DDPM: $\tilde\lambda_n\,\|\epsilon - \epsilon_\theta(\mathbf{X}^{(n)}, y, n)\|^2$.

CFG sampling rule

At inference, combine the two forward passes:

$$\tilde{\epsilon}(\mathbf{X}^{(n)}, y, n) \;=\; \omega\,\epsilon_\theta(\mathbf{X}^{(n)}, y, n)\; -\;(\omega - 1)\,\epsilon_\theta(\mathbf{X}^{(n)}, \varnothing, n),\qquad \omega > 1.$$

Use $\tilde{\epsilon}$ in place of $\epsilon_\theta$ in the DDPM / DDIM sampler.

Interpretation. Push the sample toward the label-consistent mode with weight $\omega$, then pull it back with the unconditional score at weight $\omega - 1$. Net movement toward $y$ at effective weight $1$, with the label's pull sharpened by $\omega$.

CFG in practice

Design levers

  • Label-drop probability $p_\text{drop}$
  • Guidance scale $\omega$ at sampling
  • Null-token embedding (learnable)

Trade-offs

  • Larger $\omega$: sharper but less diverse samples
  • Two forward passes per step: 2× inference cost
  • No auxiliary model to train, serve, or debug

CFG is the default guidance mechanism in production text-to-image systems (Stable Diffusion, DALL-E, Imagen). It is simple, effective, and composable with any sampler.

04

Discrete Diffusion

When the data lives on $\{1, \ldots, K\}$ instead of $\mathbb{R}^d$

What breaks in the discrete case

Continuous diffusion relies on adding Gaussian noise. Discrete data (tokens, categories) does not admit addition. "Add a little noise" is not meaningful when the state space is $\{1, \ldots, K\}$.

The natural substitute is random transitions. Instead of corrupting $x$ by a small Gaussian, we switch $x$ to a different value with some probability. Over many steps the state distribution converges to a fixed stationary law, regardless of the starting point.

Discrete diffusion replaces "additive noise" with "stochastic transitions." Everything else follows the same recipe: forward, reverse, score, loss.

Discrete-time forward: transition matrices

Let $p_t \in [0,1]^K$ be the probability vector of $X_t$. A transition matrix $Q_t \in \mathbb{R}^{K\times K}$ with entries $Q_t(y, x) = \Pr(X_{t+1} = y\,|\,X_t = x)$ propagates the state:

$$p_{t+1} = Q_t\,p_t,\quad t = 0, 1, 2, \ldots$$

For a valid noising process, we need $p_t \to \pi$ for some fixed $\pi$ regardless of $p_0$. Two standard choices are uniform and absorbing (masking).

Uniform vs. absorbing transitions

Uniform

Stay with probability $1 - \epsilon$; otherwise jump uniformly.

$$\Pr(X_{t+1}=y\,|\,X_t=x) = \begin{cases} 1-\epsilon & y = x \\[2pt] \tfrac{\epsilon}{K-1} & y \ne x \end{cases}$$

Stationary distribution is uniform on $\{1, \ldots, K\}$.

Absorbing (mask)

Non-mask tokens get replaced by the mask token $[\mathrm{M}]$ at rate $\epsilon$; mask stays mask.

$$\Pr(X_{t+1}=y\,|\,X_t=x) = \begin{cases} 1-\epsilon & y = x \ne [\mathrm{M}] \\[2pt] \epsilon & x \ne y = [\mathrm{M}] \\[2pt] 1 & x = y = [\mathrm{M}] \\[2pt] 0 & \text{otherwise} \end{cases}$$

Stationary distribution: all mass on $[\mathrm{M}]$.

Continuous-time: an ODE on probability vectors

Take $\Delta t \to 0$ and the discrete updates become a linear ODE for the probability vector:

$$\frac{dp_t}{dt} = Q_t\,p_t,\qquad p_0 = p_\text{data}.$$

The reverse rate matrix follows from Bayes: for $x \ne y$,

$$\overline{Q}_t(y, x) = \frac{p_t(y)}{p_t(x)}\,Q_t(x, y),\qquad \overline{Q}_t(x, x) = -\sum_{y \ne x}\overline{Q}_t(y, x).$$

The diagonal is fixed by mass conservation ($\sum_i p_t(i) = 1$). The novel quantity is the ratio $p_t(y)/p_t(x)$.

The discrete score is a ratio

In continuous diffusion, reverse sampling needs $\nabla_{\mathbf{x}} \log p_t(\mathbf{x})$. The discrete analog is the marginal ratio, which we also call the score:

$$s(x, t) = \begin{pmatrix} p_t(1)/p_t(x) \\ p_t(2)/p_t(x) \\ \vdots \\ p_t(K)/p_t(x) \end{pmatrix} \in [0, \infty)^K.$$

We learn it with a neural network $s_\theta(x, t)$.

Continuous: gradient of log density. Discrete: ratio of densities between neighbouring states. Same role, different object.

Score entropy loss

A squared loss on $s_\theta - p_t(y)/p_t(x)$ fails: it accepts negative estimates, which ruin the reverse rates. Lou et al. (2024, ICML Best Paper) propose the score entropy objective:

$$\mathcal{L}_\text{SE}(\theta) = \mathbb{E}_{x_t \sim p_t}\!\left[\sum_{y \ne x_t} w_{x_t, y}\left(s_\theta(x_t, t)_y - \frac{p_t(y)}{p_t(x_t)}\,\log s_\theta(x_t, t)_y + C\!\left(\frac{p_t(y)}{p_t(x_t)}\right)\right)\right],$$

with $C(a) = a(\log a - 1)$. The $\log s_\theta$ term forces the estimate to stay positive. A tractable denoising variant $\mathcal{L}_\text{DSE}$ replaces $p_t$ with $p_{t\,|\,0}$, analogous to score matching in the continuous case.

Practical sequence space

Real text lives in $\mathcal{X} = \{1, \ldots, K\}^L$. A direct transition matrix over that space has $K^L \times K^L$ entries, which is intractable.

Factored assumption. Each token is perturbed independently with the same token-level rate matrix. This implies the Hamming-sparsity condition

$$Q_t(\mathbf{y}, \mathbf{x}) = 0\quad\text{whenever}\quad d_\text{H}(\mathbf{y}, \mathbf{x}) \geq 2,$$

because more than one token changing in an infinitesimal step is negligible. The reverse process only needs $s_\theta$ on Hamming-distance-1 pairs.
