01

What the DDPM Loss Sees

Freedom to redesign the forward process

A Closer Look at the DDPM Loss

The $\epsilon$-prediction training loss per step:

$$L_{n-1} = \tilde{\lambda}_n\,\mathbb{E}_{X^{(0)},\,\epsilon}\!\left[\bigl\|\epsilon - \epsilon_\theta\bigl(\sqrt{\bar{\alpha}_n}\,X^{(0)} + \sqrt{1-\bar{\alpha}_n}\,\epsilon,\; n\bigr)\bigr\|^2\right].$$

The forward process enters only through the single-time marginal $q\bigl(X^{(n)}\,|\,X^{(0)}\bigr)$. The joint $q(X^{(1:N)}\,|\,X^{(0)})$ never appears.

The Freedom This Gives

Any forward process with

$$q(X^{(n)}\,|\,X^{(0)}) = \mathcal{N}\!\bigl(\sqrt{\bar{\alpha}_n}\,X^{(0)},\;(1-\bar{\alpha}_n)\,I\bigr)$$

yields the same $\epsilon$-prediction loss and the same trained network.

DDIM: keep the same marginals, build a non-Markovian joint, use a reverse process with tunable stochasticity down to zero.

Song, Meng & Ermon, "Denoising Diffusion Implicit Models", ICLR 2021.
02

DDIM's Non-Markovian Forward

Same marginals, tunable stochasticity

DDIM Forward Distribution

Fix schedule $\{\bar{\alpha}_n\}$ and arbitrary $\{\sigma_n \geq 0\}$:

$$q(X^{(n)}\,|\,X^{(n+1)}, X^{(0)}) = \mathcal{N}\!\left(\sqrt{\bar{\alpha}_n}\,X^{(0)} + \frac{\sqrt{1-\bar{\alpha}_n - \sigma_{n+1}^2}}{\sqrt{1-\bar{\alpha}_{n+1}}}\,\bigl(X^{(n+1)} - \sqrt{\bar{\alpha}_{n+1}}\,X^{(0)}\bigr),\;\sigma_{n+1}^2\,I\right).$$

The conditional depends on both $X^{(n+1)}$ and $X^{(0)}$: non-Markovian.

Marginals Match DDPM

By induction, $q(X^{(n)}\,|\,X^{(0)}) = \mathcal{N}\!\bigl(\sqrt{\bar{\alpha}_n}\,X^{(0)},\;(1-\bar{\alpha}_n)\,I\bigr)$ for every $n$, regardless of $\{\sigma_n\}$.

  • Any DDPM-trained network is directly usable in DDIM.
  • Training cost identical. Only the sampler differs.

Deterministic Choice $\sigma_n \equiv 0$

Setting $\sigma_n = 0$: forward conditional becomes Dirac. Given $X^{(0)}$ and one later step, every other step is determined. Chain still has the same marginals as DDPM.

Interpolation: $\sigma_n = \sqrt{\beta_n \cdot (1-\bar\alpha_n)/(1-\bar\alpha_{n+1})}$ (DDPM-like) to $\sigma_n = 0$ (deterministic) is continuous.

03

DDIM Sampling

From $\epsilon_\theta$ to samples, with tunable noise

Training: Nothing Changes

DDIM trains the same $\epsilon$-prediction network as DDPM. Given $X^{(0)}$, draw $\epsilon \sim \mathcal{N}(0, I)$ and $n \sim \text{Unif}$:

$$X^{(n)} = \sqrt{\bar{\alpha}_n}\,X^{(0)} + \sqrt{1-\bar{\alpha}_n}\,\epsilon.$$

Training is Markov (loss depends only on single-time marginals). Sampling is where the non-Markovian structure pays off.

Predicted Clean Signal

At sampling time, invert the noise equation to estimate $X^{(0)}$ from the current state:

$$\hat{X}^{(0)} = \frac{X^{(n+1)} - \sqrt{1-\bar{\alpha}_{n+1}}\,\epsilon_\theta(X^{(n+1)}, n+1)}{\sqrt{\bar{\alpha}_{n+1}}}.$$

The $\epsilon$-prediction rearranged: tells us where the network thinks the clean sample lies.

DDIM Reverse Update

Plug $\hat{X}^{(0)}$ into the DDIM forward conditional:

$$X^{(n)} = \sqrt{\bar{\alpha}_n}\,\hat{X}^{(0)} + \sqrt{1-\bar{\alpha}_n-\sigma_{n+1}^2}\,\epsilon_\theta(X^{(n+1)}, n+1) + \sigma_n\,Z^{(n)},\quad Z^{(n)}\sim\mathcal{N}(0,I).$$

Three pieces: toward data (via $\hat{X}^{(0)}$), along noise direction, optional fresh noise.

Sampling Algorithm

DDIM Sampling
  1. Sample $X^{(N)} \sim \mathcal{N}(0, I)$.
  2. For $n = N{-}1, N{-}2, \ldots, 0$:
    1. Compute $\hat{X}^{(0)} = \bigl(X^{(n+1)} - \sqrt{1-\bar{\alpha}_{n+1}}\,\epsilon_\theta(X^{(n+1)}, n+1)\bigr)/\sqrt{\bar{\alpha}_{n+1}}$.
    2. If $\sigma_n > 0$: sample $Z^{(n)}\sim\mathcal{N}(0,I)$, else skip.
    3. $X^{(n)} = \sqrt{\bar{\alpha}_n}\,\hat{X}^{(0)} + \sqrt{1-\bar{\alpha}_n-\sigma_{n+1}^2}\,\epsilon_\theta(X^{(n+1)}, n+1) + \sigma_n\,Z^{(n)}$.

$\sigma_n \equiv 0$: Probability-Flow ODE

When $\sigma_n = 0$, the reverse step is deterministic. Continuous-time limit:

$$\frac{dX_t}{dt} = f(X_t, t) - \frac{g^2(t)}{2}\,\nabla_{\mathbf{x}} \log p_t(X_t).$$

This ODE has the same marginals $\{p_t\}$ as the forward SDE. DDIM with $\sigma_n = 0$ is a specific discretization for VP diffusion.

Stochastic forward, deterministic generative process. Same marginals. Score does all the work.

04

Consequences of Determinism

Why DDIM matters in practice

Three Benefits

Fewer steps

Large strides on the index axis. 50 steps often match 1000-step DDPM quality.

Inversion

Run the deterministic generator backwards: recover the latent noise from an image. Used for editing and attribution.

Latent interpolation

Interpolating noise latents yields smooth, meaningful paths in image space.

All three hinge on: $X^{(0)}$ is a deterministic function of $X^{(N)}$ under $\sigma_n = 0$.

Lecture Recap

  • DDPM $\epsilon$-loss depends only on $q(X^{(n)}\,|\,X^{(0)})$, not the joint.
  • DDIM preserves that marginal via a non-Markovian forward parameterized by $\{\sigma_n\}$.
  • $\sigma_n \equiv 0$: reverse is deterministic, discretizes the probability-flow ODE.
  • Training shared. Sampling is a solver choice.
  • Wins: fewer steps, invertibility, clean latent interpolation.

Check It Yourself

  1. Show by induction that DDIM forward satisfies $q(X^{(n)}\,|\,X^{(0)}) = \mathcal{N}\!\bigl(\sqrt{\bar{\alpha}_n}\,X^{(0)},\;(1-\bar{\alpha}_n)\,I\bigr)$ for any $\{\sigma_n\}$.
  2. Show that $\sigma_n^2 = \beta_n\,(1-\bar{\alpha}_n)/(1-\bar{\alpha}_{n+1})$ recovers the DDPM reverse update.
  3. Show that the $\sigma_n = 0$ DDIM update is a valid explicit discretization of the probability-flow ODE for VP diffusion.
