Slide — Lossless is not enough

For a discrete random variable on a finite alphabet, Shannon's source coding theorem says we can encode i.i.d. samples at rate arbitrarily close to $H(X)$ bits per symbol. For continuous random variables, the analogous quantity is the differential entropy $h(X)$, but $h$ is not the number of bits we need to store $X$ — in fact $H(X)=\infty$ for any continuous $X$, because describing a real number to infinite precision takes infinitely many bits.

This is not a defect of the source — it is the price of asking for perfect reconstruction of a real-valued object. The lossy formulation accepts a controlled error in exchange for a finite rate, and the question becomes: how small must this error be for a given rate, in the asymptotic regime of long blocks?

Slide — Distortion functions

The squared error and Hamming distortions are by far the most common, but the choice of distortion function is itself a modeling choice. For example, in image compression a perceptual metric (e.g. SSIM) gives wildly different optimal codes than MSE, because the human visual system is more sensitive to certain spatial frequencies than others.

For this lecture we mostly work with the three single-letter distortions on the slide. Single-letter means $d^{(n)}(x^n,\hat x^n)=\frac{1}{n}\sum_{i=1}^n d(x_i,\hat x_i)$ — the per-block distortion is the average of per-coordinate distortions. Most theorems below assume this structure.

Slide — Block code

Note that $R$ is measured in bits per source symbol, not bits per block. A block code of size $2^{nR}$ stores an index using $nR$ bits to describe a block of $n$ source samples. If $R$ is in nats instead, the message set has size $e^{nR}$.

The encoder $f$ and decoder $g$ are arbitrary deterministic maps. They may be jointly designed; the only thing fixed is the rate $R$ and the requirement that the index alphabet has size at most $2^{nR}$. We will see later that randomness in the codebook design (random coding) is a powerful proof technique even when the final code is deterministic.

Slide — Achievable region

The $\varepsilon$ in the definition matters. We allow the actual rate to exceed $R$ by any positive $\varepsilon$ and the actual distortion to exceed $D$ by any positive $\varepsilon$, provided $n$ can be chosen large enough. In the limit $\varepsilon\downarrow 0$, $n\to\infty$, this becomes "the" rate–distortion function. Without these slack parameters, the operational $R(D)$ would not be a clean function of $D$.

The achievable region is closed and "upper-monotone": if $(R,D)$ is achievable, so is $(R',D')$ for any $R'\ge R$ and $D'\ge D$. The function $R(D)$ is non-increasing, convex, and continuous on $D\in (D_{\min}, D_{\max})$. The convexity comes from a time-sharing argument: a code that mixes a $(R_1,D_1)$ code with a $(R_2,D_2)$ code achieves $(\lambda R_1+(1-\lambda)R_2, \lambda D_1+(1-\lambda)D_2)$ for any $\lambda\in[0,1]$.

Slide — Distortion balls cover the typical set

Lossless coding asks: how many distinct strings carry typical-set probability mass? The answer is roughly $2^{nH(X)}$, leading to rate $H(X)$.

Lossy coding asks instead: how many distortion-$D$ balls cover the typical set? Each ball "absorbs" all typical strings within distortion $D$ of its center, and a single index is enough to point to the ball. The smallest log-count of such balls is exactly $R(D)$. This is more than an analogy — the achievability proof literally counts random codewords against typical-set volumes.

Slide — Heights toy and $R=0$

If we send no bits, the decoder must output a fixed deterministic value $c$. Under squared error, $\mathbb{E}[(X-c)^2]$ is minimized over $c$ by the mean (this is one of the few useful facts in statistics that survives all generalizations). The minimum value is the variance.

So already at $R=0$ we have a non-trivial code: output $c=\mu$. Any source with finite variance has $R(\sigma^2)=0$ — we don't need to send anything if we are allowed distortion at least the variance.

Slide — $R=1$ Gaussian computation

For a Gaussian, the optimal 1-bit threshold is at the median, which is $\mu$. Conditional on $X>\mu$, $X$ has a half-Gaussian distribution. Its mean is

\[ \mathbb{E}[X\mid X>\mu]=\mu + \sigma\sqrt{\frac{2}{\pi}}. \]

By symmetry, the lower-cell mean is $\mu-\sigma\sqrt{2/\pi}$. The remaining variance within each cell is $\sigma^2(1-2/\pi)$, and by symmetry the total expected squared error is the same:

\[ D(1)=\sigma^2\!\left(1-\frac{2}{\pi}\right)\approx 0.363\sigma^2. \]

Compare to $D(0)=\sigma^2$. One bit cuts distortion by a factor of $\approx 2.75$. This is far from the rate–distortion limit $D(1)=\sigma^2 \cdot 4^{-1}=0.25\sigma^2$ achievable in the limit of long blocks — scalar quantization leaves a lot on the table.

Slide — Lloyd–Max

The Lloyd–Max algorithm is the special case of $k$-means in 1D, applied to the source distribution rather than to a sample. The two updates are necessary conditions for local optimality; they are not jointly sufficient for global optimality, and Lloyd–Max can converge to local optima that depend on the initial thresholds. In practice this matters less for unimodal sources (Gaussian, Laplace) than for multimodal ones.

Each step monotonically decreases the squared-error objective:

  1. Centroid update is the global minimizer of $\sum_i \int_{C_i}(x-c_i)^2 p(x)\,dx$ over $\{c_i\}$ at fixed cells.
  2. Boundary update is the global minimizer of the same objective over $\{\tau_i\}$ at fixed reps, because the optimal Voronoi tessellation under squared error places thresholds at midpoints of adjacent representatives.

So the algorithm is a coordinate descent on a non-negative bounded objective, hence convergent.

Slide — Vector vs. scalar quantization

The slide claim that VQ beats scalar quantization even for i.i.d. components has two distinct sources:

  1. Cell shape gain ("space-filling gain"): the disk minimizes second moment among all 2D shapes of fixed area; scalar quantizers force rectangular cells. This effect is small in 2D (about 8%) but grows with dimension.
  2. Shape gain from non-uniform sources: a vector quantizer can put more codepoints where the source is denser. Scalar quantizers do this too, per coordinate, but they cannot adapt to joint structure (e.g. the typical shell of a high-dim Gaussian).

In high dimension, both effects together let VQ approach the rate–distortion bound; scalar quantization is bounded away.

Slide — What are we optimizing over?

The information rate–distortion function $R^{(I)}(D)$ is defined as a constrained minimization over a probability object. It is worth being explicit about what is fixed, what is free, and what is derived:

  • Fixed. The source marginal $P_X$ (handed to us by the world), the distortion measure $d(\cdot,\cdot)$, and the budget $D$.
  • Free. The conditional distribution $P_{\widehat X\mid X}$ — equivalently, a stochastic transition kernel from the source alphabet to the reproduction alphabet. Each row $P_{\widehat X\mid X}(\cdot\mid x)$ is a probability vector. This object is also called the test channel.
  • Derived. Once $P_{\widehat X\mid X}$ is chosen, the joint $P_{X,\widehat X}=P_X\,P_{\widehat X\mid X}$, the output marginal $P_{\widehat X}=\sum_x P_X(x)P_{\widehat X\mid X}(\cdot\mid x)$, and the mutual information $I(X;\widehat X)$ are all determined.

The optimization is a convex program: $I(X;\widehat X)$ is a convex function of $P_{\widehat X\mid X}$ when $P_X$ is held fixed (a standard fact in information theory). The distortion constraint $\mathbb{E}\,d \le D$ is linear in $P_{\widehat X\mid X}$. Hence Lagrangian / KKT methods apply, and the solution can be characterized by stationarity conditions.

The Blahut–Arimoto algorithm is one numerical procedure for solving this optimization for arbitrary discrete sources; for many continuous sources of interest (Gaussian, Laplacian) the solution can be found in closed form, as we will see in Lecture 2.

Slide — Solving the optimization (binary)

For $X\sim\mathrm{Bern}(1/2)$ under Hamming distortion, the test channel is a $2\times 2$ row-stochastic matrix with two free entries:

\[ P_{\widehat X\mid X} = \begin{pmatrix} 1-\alpha_0 & \alpha_0 \\ \alpha_1 & 1-\alpha_1 \end{pmatrix}, \qquad \mathbb{E}\,d_H = \tfrac{\alpha_0+\alpha_1}{2}. \]

By the symmetry of source and distortion under $X\leftrightarrow 1-X$, the optimum is symmetric: $\alpha_0=\alpha_1=\alpha$. (More formally, if $(\alpha_0^\star,\alpha_1^\star)$ is optimal then so is $(\alpha_1^\star,\alpha_0^\star)$ by relabeling, and the average $(\bar\alpha,\bar\alpha)$ is at least as good by convexity.) With this reduction:

\[ \widehat X\sim\mathrm{Bern}(1/2),\qquad H(\widehat X\mid X) = H_b(\alpha),\qquad I(X;\widehat X) = 1 - H_b(\alpha). \]

Since $1-H_b(\alpha)$ is strictly decreasing on $\alpha\in[0,1/2]$, the constrained minimum is attained at the boundary $\alpha=D$ (assuming $D\le 1/2$). This gives $R^{(I)}(D)=1-H_b(D)$ and the optimal joint $P^\star_{X,\widehat X}$ described by $X=\widehat X\oplus Z$ with $\widehat X\sim\mathrm{Bern}(1/2)$ and $Z\sim\mathrm{Bern}(D)$ independent.

Note that the optimal output marginal $P^\star_{\widehat X}=\mathrm{Bern}(1/2)$ matches the source marginal here. This is a coincidence of the binary symmetric setup; for general sources the optimal $P^\star_{\widehat X}$ does not equal $P_X$ (e.g. for Laplacian sources $P^\star_{\widehat X}$ has a Dirac atom at zero — see Lecture 2).

Slide — Suboptimal codebook (wrong marginal)

The achievability theorem says: draw codewords iid from $P^\star_{\widehat X}$ and rate $R>R(D)$ suffices. What if we instead drew codewords from a different product distribution — say $\mathrm{Bern}(p)^n$ with $p\ne 1/2$? The codebook still works, but at a strictly worse rate.

Fix a typical $X^n$ with $k\approx n/2$ ones. The bits of the disagreement vector $X^n\oplus\widehat X^n$ are independent: rate $p$ at zeros-positions of $X^n$, rate $1-p$ at ones-positions. By Sanov's theorem, the probability that the empirical disagreement rate is at most $D$ decays exponentially with rate

\[ \rho(p,D) \;=\; \min_{\substack{q_0,q_1\\ (q_0+q_1)/2\,\le\,D}} \tfrac{1}{2}\bigl[D_{\mathrm{KL}}(q_0\|p) + D_{\mathrm{KL}}(q_1\|1-p)\bigr], \]

where $q_0$ is the optimal exponent on zeros-positions and $q_1$ on ones-positions. By Lagrangian stationarity, $q_0$ and $q_1$ satisfy

\[ \frac{q_0(1-p)}{p(1-q_0)} \;=\; \frac{q_1\,p}{(1-p)(1-q_1)} \qquad\text{and}\qquad \tfrac{q_0+q_1}{2} = D. \]

For $p=1/2$ the system is symmetric: $q_0=q_1=D$, $\rho(1/2,D)=1-H_b(D)=R(D)$. For $p\ne 1/2$ the symmetry is broken — on zeros-positions a flip is the unlikely event ($p$ small), so $q_0$ stays small; on ones-positions a "flip" means agreeing with $X_i=1$ when the codeword has bias $p$ for ones, which is the unlikely event when $p<1/2$, so $q_1$ is pushed up.

Worked numbers for $D=1/4$:

  • $p=1/4$: $16q_0^2+12q_0-1=0$ gives $q_0\approx 0.0757$, $q_1\approx 0.4243$. Plugging in: $\rho\approx 0.246$ bits, vs $R(D)\approx 0.189$ bits. Codebook size threshold rises from $2^{n\cdot 0.189}\approx 23$ to $2^{n\cdot 0.246}\approx 60$ at $n=24$.
  • $p=1/8$: $96q_0^2+52q_0-1=0$ gives $q_0\approx 0.0186$, $q_1\approx 0.4814$. $\rho\approx 0.381$ bits. Codebook threshold $\approx 565$ at $n=24$ — an order of magnitude over the optimal $\sim 23$.

The penalty grows as $p$ moves away from $1/2$, and at $p=0$ or $p=1$ the codebook becomes a deterministic constant string: the only disagreement-rate distribution achievable is $(q_0,q_1)=(p,1-p)$, which gives mean disagreement $1/2$ and no slack to dip below $D<1/2$ — $\rho\to\infty$.

Operationally: every codeword law that is not the optimal $P^\star_{\widehat X}$ wastes bits. The information – theoretic optimization $\arg\inf I(X;\widehat X)$ is exactly the codebook law that minimizes the rate needed to find a match.

Slide — The mutual information formula

Shannon's theorem is in two parts:

  • Achievability: for any test channel $P_{\widehat X\mid X}$ achieving distortion $\le D$ and information $I$, every rate $R>I$ is achievable. We sketch this via random coding on the next slides.
  • Converse: no code achieves rate below $R^{(I)}(D)$. This is the inequality $nR\ge I(X^n;\widehat X^n)\ge n\inf I(X;\widehat X)$, where the last step is by the i.i.d. structure and a single-letterization step that needs the convexity of mutual information in the conditional.

For continuous sources we replace $H$ with $h$ (differential entropy) and the formulas continue to hold.

Slide — Random coding rare-event bound

Joint typicality with respect to $P_{X,\widehat X}$ asks for the empirical joint distribution to be close (in some norm) to the target joint distribution. By the asymptotic equipartition property,

\[ \Pr_{P_X^{\otimes n}\,\otimes\, P_{\widehat X}^{\otimes n}}\bigl[(X^n,\widehat X^n)\in T_\varepsilon^{(n)}\bigr]\;\approx\;2^{-nI(X;\widehat X)}. \]

The "approx" hides $\varepsilon$ slack and lower-order terms. The bound is exponentially tight for large $n$.

The reason is geometric: the joint typical set has volume $\approx 2^{n H(X,\widehat X)}$, while the product of the marginals has volume $\approx 2^{n H(X)}\cdot 2^{n H(\widehat X)}=2^{n(H(X)+H(\widehat X))}$. Their ratio is $2^{-nI(X;\widehat X)}$.

Slide — Search creates dependence

The key conceptual point: the encoder is allowed to look at $X^n$ when picking $m$. If we had to commit to an index without seeing the source, we could not do better than chance, and the average distortion would be $\mathbb{E}[d(X,\widehat X)]$ where $X$ and $\widehat X$ are independent — usually no better than the $R=0$ bound.

By searching, we condition on the rare event that some particular codeword is jointly typical with $X^n$. Conditioning on a rare event with the right structure pushes the conditional joint towards the desired $P_{X,\widehat X}$. This is the same phenomenon as posterior concentration in Bayesian statistics: a rare event with the right shape effectively reweights the prior into the posterior.

Slide — Binary toy details

The bit strings on the slide are illustrative and were chosen so that one of the four "quick draws" lies relatively far from $X^n$ (Hamming distance 14) and the eventual successful candidate (#14 of 64) lies exactly on the radius-6 ball. In a real run with a fresh random seed the exact distances will differ, but the qualitative behavior is robust:

  • Most random Bernoulli$(1/2)$ codewords lie at Hamming distance close to $n/2$ from any fixed source $X^n$.
  • The fraction within distance $nD$ of $X^n$ is $|B(X^n,nD)|/2^n\approx 2^{-n(1-H_b(D))}$, which is the inverse of $2^{nR(D)}$.
  • So a codebook of size $2^{nR}$ contains a successful match with probability $1-(1-2^{-nR(D)})^{2^{nR}}\approx 1-\exp(-2^{n(R-R(D))})$, which is small if $R<R(D)$ and close to 1 if $R>R(D)$.

For $n=24, D=1/4$: the threshold rate is $R(D)\approx 0.189$ bits/symbol, so we need $2^{0.189\cdot 24}\approx 23$ codewords to have a $\sim 63\%$ chance of finding a match (at the threshold), and $\sim 64$ codewords to be almost sure. The slide's table at 64 codewords is the "almost sure" regime.

Slide — Volume counting threshold

The Hamming-ball volume formula is asymptotic:

\[ |B(0,nD)|=\sum_{k=0}^{nD}\binom{n}{k}\;\le\;2^{nH_b(D)},\qquad D\le 1/2, \]

and the bound is exponentially tight: $|B(0,nD)|\sim 2^{nH_b(D)}/\sqrt{n}$ up to polynomial factors. Plugging into the cover bound gives the threshold rate $R(D)=1-H_b(D)$.

For non-uniform Bernoulli sources (Bernoulli$(p)$ with $p\ne 1/2$), the analogous threshold is $R(D)=H_b(p)-H_b(D)$ for $D\le p$, which reflects the smaller entropy of a biased source.

Slide — Worked example: biased binary

The binary symmetric source ($p=1/2$) is the simplest case but somewhat misleading: source and codebook share the same marginal $\mathrm{Bern}(1/2)$. The biased case $p<1/2$ breaks this coincidence and is the cleanest illustration that the optimal codebook law is different from the source law.

Lower bound (converse).

For any $P_{X,\widehat X}$ with $X\sim\mathrm{Bern}(p)$ and $\mathbb E[d_H(X,\widehat X)]=\Pr(X\ne\widehat X)\le D$,

\[ \begin{aligned} I(X;\widehat X) &= H(X) - H(X\mid\widehat X) \\ &= H_b(p) - H(X\oplus\widehat X\mid\widehat X) \\ &\ge H_b(p) - H(X\oplus\widehat X) \\ &\ge H_b(p) - H_b\!\bigl(\Pr(X\ne\widehat X)\bigr) \\ &\ge H_b(p) - H_b(D), \end{aligned} \]

where the inequalities use (i) conditioning reduces entropy, (ii) the binary entropy is concave and increasing on $[0,1/2]$, and (iii) the distortion constraint $\Pr(X\ne\widehat X)\le D\le p\le 1/2$.

Achievability (matching upper bound).

Set $X=\widehat X\oplus Z$ with $\widehat X\sim\mathrm{Bern}(q)$, $Z\sim\mathrm{Bern}(D)$, $\widehat X\perp Z$. The constraint $\Pr(X=1)=p$ gives one equation in one unknown:

\[ q(1-D) + (1-q)D = p \;\Longrightarrow\; q = \frac{p-D}{1-2D}. \]

Note $q\le p$ for $D\ge 0$, with equality only at $D=0$ (no compression). The mutual information is

\[ I(X;\widehat X) = H(X) - H(X\mid\widehat X) = H_b(p) - H(Z) = H_b(p) - H_b(D), \]

matching the lower bound. Hence $R(D)=H_b(p)-H_b(D)$ for $0\le D\le p$, and $R(D)=0$ for $D\ge p$ (output the constant $\widehat X=0$ and accept distortion $p$).

Why the codeword law differs from the source law.

The optimal output marginal is $P^\star_{\widehat X}=\mathrm{Bern}(q)$ with $q=(p-D)/(1-2D)$. For $D>0$ this is strictly less than $p$ — the codewords are sparser than the source. Intuitively, the encoder relies on the noise $Z$ (rate $D$) to flip a fraction $D$ of the bits up to the source rate $p$; the codeword only needs to carry the "remaining" ones, which are scarcer. This is the simplest example of a phenomenon that becomes dramatic for the Laplacian source in Lecture 2, where the optimal output marginal develops a Dirac atom at zero (sparse reproduction) even though the source itself has full support.

Numerics for $p=0.2,D=0.1$.

$H_b(0.2)=-0.2\log_2 0.2 - 0.8\log_2 0.8 \approx 0.7219$ bits. $H_b(0.1)\approx 0.4690$ bits. $R(D)\approx 0.2529$ bits/symbol. The optimal $\widehat X^\star\sim\mathrm{Bern}(1/8)$. Codebook size for block length $n$: $M=\lceil 2^{n\cdot 0.253}\rceil$.

For comparison, the symmetric case ($p=1/2,D=0.1$) would give $R(D)=1-H_b(0.1)\approx 0.531$ bits — more than double. A biased source has lower entropy and is therefore easier to compress, which the formula $H_b(p)-H_b(D)$ captures cleanly.

Slide — Forbidden reproduction (infinite distortion)

This example illustrates what happens when one direction of the distortion is infinite — an absolute prohibition on certain reproduction choices — while the symmetric source is unchanged. The asymmetry forces structural changes in the optimal joint.

Why the joint becomes triangular.

The expected distortion is

\[ \mathbb{E}[d(X,\widehat X)] = \sum_{x,\hat x} p(x,\hat x)\,d(x,\hat x). \]

If $d(0,1)=\infty$ and we want a finite expected distortion, the only way is to set the coefficient $p(0,1) = 0$ exactly. Any positive mass there would produce $0\cdot\infty = \infty$ (or, more carefully: the limit makes the constraint infeasible). So the joint matrix has a hard zero in the $(0,1)$ corner.

The other three entries must sum to $1$ and respect the source marginal $p(X{=}0)=p(X{=}1)=1/2$. After accounting for these constraints, only one parameter remains, namely $D = p(1,0)$, and the joint takes the triangular form on the slide.

The output marginal is biased.

The output marginal is $\Pr(\widehat X{=}0) = 1/2 + D$ and $\Pr(\widehat X{=}1) = 1/2 - D$. For any $D\in(0,1/2)$ this is strictly biased toward $0$ — intuitively, the encoder “dumps” the lossy bits onto the safe side ($\widehat X{=}0$ is a permitted reproduction for both source values), and only carefully sets $\widehat X{=}1$ when the source is actually $1$.

The mutual information.

Using $H(X)=1$ and computing the conditional entropy:

\[ \begin{aligned} H(X\mid\widehat X{=}0) &= H_b\!\Bigl(\tfrac{1/2}{1/2+D}\Bigr), \\ H(X\mid\widehat X{=}1) &= 0 \quad \text{(deterministic)}, \\ H(X\mid\widehat X) &= \bigl(\tfrac{1}{2}+D\bigr)\,H_b\!\Bigl(\tfrac{1/2}{1/2+D}\Bigr) \;+\; \bigl(\tfrac{1}{2}-D\bigr)\cdot 0. \end{aligned} \]

Subtracting from $H(X)=1$ gives the formula on the slide. After algebraic simplification using $H_b(q) = -q\log q - (1-q)\log(1-q)$:

\[ R(D) = \tfrac{1}{2} \;-\; \bigl(\tfrac{1}{2}+D\bigr)\log\!\bigl(\tfrac{1}{2}+D\bigr) \;+\; D\log D. \]

The asymmetric (“forbidden”) version always requires more bits at the same distortion. The geometric reason: the test channel has a hard wall on one side, so the encoder can’t use both halves of the reproduction alphabet symmetrically.

Why this matters.

This is the simplest concrete example where $P^\star_{\widehat X} \ne P_X$ even though $P_X$ is uniform. The classical “binary symmetric” case has $P^\star_{\widehat X}=\mathrm{Bern}(1/2)=P_X$ as a coincidence; the forbidden-reproduction case breaks the coincidence cleanly. It also sets up intuition for the Laplacian case (Lecture 2), where the optimal output develops a hard atom at zero — the continuous analogue of dumping mass onto a “safe” reproduction.

Slide — Uniform ternary source & the Shannon lower bound

This example introduces the Shannon lower bound in a discrete setting: a tight, conditional-entropy-style bound on $R(D)$ that doesn’t require any explicit construction. For the uniform ternary source under Hamming distortion the bound is tight; for non-uniform sources it is generally a strict bound but still useful.

The chain of inequalities.

Let $V$ be the reproduction. The four steps in the slide’s derivation:

  1. (i): $I(U;V) = H(U) - H(U\mid V)$ by definition.
  2. (ii): $H(U\mid V) = H(U\ominus V\mid V)$ because, conditional on $V=v$, the map $u\mapsto u\ominus v$ is a bijection on $\{0,1,2\}$ — it just relabels symbols, leaving the conditional entropy unchanged.
  3. (iii): $H(U\ominus V \mid V) \le H(U\ominus V)$ because conditioning never increases entropy.
  4. (iv): $H(U\ominus V) \le \phi(D)$ because the marginal of $Z := U\ominus V$ has $\Pr(Z\ne 0) = \Pr(U\ne V) = \mathbb{E}[d_H(U,V)] \le D$, so it lies in the feasible set defining $\phi$.

The chain holds for every joint $P_{U,V}$ in the feasible set, so the infimum over feasible joints is also bounded below by $H(U)-\phi(D)$. That gives $R(D) \ge H(U) - \phi(D)$.

Computing $\phi(D)$ explicitly.

$\phi(D)$ asks: among ternary distributions $Z$ with $\Pr(Z\ne 0) \le D$, which has the largest entropy? Write the law as $(p_0, p_1, p_2)$ with $p_1+p_2 \le D$ and $p_0 = 1-(p_1+p_2)$. Concavity of entropy says we should

  • Push the constraint to the boundary: $p_1+p_2 = D$ exactly (giving $p_0 = 1-D$).
  • Distribute the remaining mass $D$ uniformly over the two symbols $\{1,2\}$: $p_1 = p_2 = D/2$.

Plugging in:

\[ \begin{aligned} \phi(D) &= -(1-D)\log(1-D) - 2\cdot\tfrac{D}{2}\log\tfrac{D}{2} \\ &= H_b(D) - D\log\tfrac{1}{2} \\ &= H_b(D) + D. \end{aligned} \]

This formula holds for $0 \le D \le 2/3$. For $D \ge 2/3$ the unconstrained max is achieved at uniform $Z$ with $H(Z) = \log 3$, so $\phi(D) = \log 3$ and $R(D) = 0$.

Why the bound is tight for uniform $U$.

Tightness requires simultaneous equality in (iii) and (iv):

  • (iii) is tight iff $U \ominus V$ is independent of $V$.
  • (iv) is tight iff $U \ominus V$ has the maximizing law $(1-D, D/2, D/2)$.

Construction: pick $V$ uniform on $\{0,1,2\}$, $Z_D \sim (1-D, D/2, D/2)$ independent of $V$, and set $U = V \oplus Z_D$. Then

  • $U$ is uniform: convolving uniform with anything on a finite group stays uniform (this is special to uniform $U$ and is the reason the bound is tight in this case).
  • $U \ominus V = Z_D$ is independent of $V$ — (iii) tight.
  • $Z_D$ has the maximizing law — (iv) tight.

Therefore $R(D) = \log 3 - \phi(D) = \log 3 - H_b(D) - D$ on $[0, 2/3]$.

Generalization.

For a $k$-ary uniform source under Hamming distortion, the same argument gives $R(D) = \log k - H_b(D) - D\log(k-1)$ for $0\le D\le 1-1/k$, by setting $\phi(D) = H_b(D) + D\log(k-1)$ — spread the mismatch mass uniformly over the $k-1$ non-zero symbols. At $k=2$ this recovers $R(D) = 1 - H_b(D)$ (the binary symmetric case), and at $k=3$ we get $R(D) = \log 3 - H_b(D) - D$ with $\log(k-1) = \log 2 = 1$. So the ternary formula is one slot in a family.

For non-uniform sources the Shannon lower bound is generally not tight; an additional term involving $D_{\mathrm{KL}}$ from a balancing distribution appears. The uniform case is special because the convolution argument preserves uniformity exactly.

Looking ahead

Lecture 2 makes the rate–distortion function explicit for two important continuous sources:

  • Gaussian under squared error: $R(D)=\frac{1}{2}\log(\sigma^2/D)$, with the backward Gaussian test channel and the Shannon lower bound (Gaussian is the worst case at fixed variance).
  • Laplacian under absolute error: $R(D)=-\log(\lambda D)$, with the surprising consequence that the optimal reproduction has a Dirac atom at zero — pruning falls out of the rate–distortion theorem.

Both will be derived from the mutual-information formula, using the maximum-entropy property of Gaussians and a clever auxiliary-density trick for Laplace.
