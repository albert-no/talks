Slide — Lossless is not enough

For a discrete random variable on a finite alphabet, Shannon's source coding theorem says we can encode i.i.d. samples at rate arbitrarily close to $H(X)$ bits per symbol. For continuous random variables, the analogous quantity is the differential entropy $h(X)$, but $h$ is not the number of bits we need to store $X$ — in fact $H(X)=\infty$ for any continuous $X$, because describing a real number to infinite precision takes infinitely many bits.

This is not a defect of the source — it is the price of asking for perfect reconstruction of a real-valued object. The lossy formulation accepts a controlled error in exchange for a finite rate, and the question becomes: how small must this error be for a given rate, in the asymptotic regime of long blocks?

Slide — Distortion functions

The squared error and Hamming distortions are by far the most common, but the choice of distortion function is itself a modeling choice. For example, in image compression a perceptual metric (e.g. SSIM) gives wildly different optimal codes than MSE, because the human visual system is more sensitive to certain spatial frequencies than others.

For this lecture we mostly work with the three single-letter distortions on the slide. Single-letter means $d^{(n)}(x^n,\hat x^n)=\frac{1}{n}\sum_{i=1}^n d(x_i,\hat x_i)$ — the per-block distortion is the average of per-coordinate distortions. Most theorems below assume this structure.

Slide — Block code

Note that $R$ is measured in bits per source symbol, not bits per block. A block code of size $2^{nR}$ stores an index using $nR$ bits to describe a block of $n$ source samples. If $R$ is in nats instead, the message set has size $e^{nR}$.

The encoder $f$ and decoder $g$ are arbitrary deterministic maps. They may be jointly designed; the only thing fixed is the rate $R$ and the requirement that the index alphabet has size at most $2^{nR}$. We will see later that randomness in the codebook design (random coding) is a powerful proof technique even when the final code is deterministic.

Slide — Achievable region

The $\varepsilon$ in the definition matters. We allow the actual rate to exceed $R$ by any positive $\varepsilon$ and the actual distortion to exceed $D$ by any positive $\varepsilon$, provided $n$ can be chosen large enough. In the limit $\varepsilon\downarrow 0$, $n\to\infty$, this becomes "the" rate–distortion function. Without these slack parameters, the operational $R(D)$ would not be a clean function of $D$.

The achievable region is closed and "upper-monotone": if $(R,D)$ is achievable, so is $(R',D')$ for any $R'\ge R$ and $D'\ge D$. The function $R(D)$ is non-increasing, convex, and continuous on $D\in (D_{\min}, D_{\max})$. The convexity comes from a time-sharing argument: a code that mixes a $(R_1,D_1)$ code with a $(R_2,D_2)$ code achieves $(\lambda R_1+(1-\lambda)R_2, \lambda D_1+(1-\lambda)D_2)$ for any $\lambda\in[0,1]$.

Slide — Distortion balls cover the typical set

Lossless coding asks: how many distinct strings carry typical-set probability mass? The answer is roughly $2^{nH(X)}$, leading to rate $H(X)$.

Lossy coding asks instead: how many distortion-$D$ balls cover the typical set? Each ball "absorbs" all typical strings within distortion $D$ of its center, and a single index is enough to point to the ball. The smallest log-count of such balls is exactly $R(D)$. This is more than an analogy — the achievability proof literally counts random codewords against typical-set volumes.

Slide — Heights toy and $R=0$

If we send no bits, the decoder must output a fixed deterministic value $c$. Under squared error, $\mathbb{E}[(X-c)^2]$ is minimized over $c$ by the mean (this is one of the few useful facts in statistics that survives all generalizations). The minimum value is the variance.

So already at $R=0$ we have a non-trivial code: output $c=\mu$. Any source with finite variance has $R(\sigma^2)=0$ — we don't need to send anything if we are allowed distortion at least the variance.

Slide — $R=1$ Gaussian computation

For a Gaussian, the optimal 1-bit threshold is at the median, which is $\mu$. Conditional on $X>\mu$, $X$ has a half-Gaussian distribution. Its mean is

\[ \mathbb{E}[X\mid X>\mu]=\mu + \sigma\sqrt{\frac{2}{\pi}}. \]

By symmetry, the lower-cell mean is $\mu-\sigma\sqrt{2/\pi}$. The remaining variance within each cell is $\sigma^2(1-2/\pi)$, and by symmetry the total expected squared error is the same:

\[ D(1)=\sigma^2\!\left(1-\frac{2}{\pi}\right)\approx 0.363\sigma^2. \]

Compare to $D(0)=\sigma^2$. One bit cuts distortion by a factor of $\approx 2.75$. This is far from the rate–distortion limit $D(1)=\sigma^2 \cdot 4^{-1}=0.25\sigma^2$ achievable in the limit of long blocks — scalar quantization leaves a lot on the table.

Slide — Lloyd–Max

The Lloyd–Max algorithm is the special case of $k$-means in 1D, applied to the source distribution rather than to a sample. The two updates are necessary conditions for local optimality; they are not jointly sufficient for global optimality, and Lloyd–Max can converge to local optima that depend on the initial thresholds. In practice this matters less for unimodal sources (Gaussian, Laplace) than for multimodal ones.

Each step monotonically decreases the squared-error objective:

  1. Centroid update is the global minimizer of $\sum_i \int_{C_i}(x-c_i)^2 p(x)\,dx$ over $\{c_i\}$ at fixed cells.
  2. Boundary update is the global minimizer of the same objective over $\{\tau_i\}$ at fixed reps, because the optimal Voronoi tessellation under squared error places thresholds at midpoints of adjacent representatives.

So the algorithm is a coordinate descent on a non-negative bounded objective, hence convergent.

Slide — Vector vs. scalar quantization

The slide claim that VQ beats scalar quantization even for i.i.d. components has two distinct sources:

  1. Cell shape gain ("space-filling gain"): the disk minimizes second moment among all 2D shapes of fixed area; scalar quantizers force rectangular cells. This effect is small in 2D (about 8%) but grows with dimension.
  2. Shape gain from non-uniform sources: a vector quantizer can put more codepoints where the source is denser. Scalar quantizers do this too, per coordinate, but they cannot adapt to joint structure (e.g. the typical shell of a high-dim Gaussian).

In high dimension, both effects together let VQ approach the rate–distortion bound; scalar quantization is bounded away.

Slide — The mutual information formula

Shannon's theorem is in two parts:

  • Achievability: for any test channel $P_{\widehat X\mid X}$ achieving distortion $\le D$ and information $I$, every rate $R>I$ is achievable. We sketch this via random coding on the next slides.
  • Converse: no code achieves rate below $R^{(I)}(D)$. This is the inequality $nR\ge I(X^n;\widehat X^n)\ge n\inf I(X;\widehat X)$, where the last step is by the i.i.d. structure and a single-letterization step that needs the convexity of mutual information in the conditional.

For continuous sources we replace $H$ with $h$ (differential entropy) and the formulas continue to hold.

Slide — Random coding rare-event bound

Joint typicality with respect to $P_{X,\widehat X}$ asks for the empirical joint distribution to be close (in some norm) to the target joint distribution. By the asymptotic equipartition property,

\[ \Pr_{P_X^{\otimes n}\,\otimes\, P_{\widehat X}^{\otimes n}}\bigl[(X^n,\widehat X^n)\in T_\varepsilon^{(n)}\bigr]\;\approx\;2^{-nI(X;\widehat X)}. \]

The "approx" hides $\varepsilon$ slack and lower-order terms. The bound is exponentially tight for large $n$.

The reason is geometric: the joint typical set has volume $\approx 2^{n H(X,\widehat X)}$, while the product of the marginals has volume $\approx 2^{n H(X)}\cdot 2^{n H(\widehat X)}=2^{n(H(X)+H(\widehat X))}$. Their ratio is $2^{-nI(X;\widehat X)}$.

Slide — Search creates dependence

The key conceptual point: the encoder is allowed to look at $X^n$ when picking $m$. If we had to commit to an index without seeing the source, we could not do better than chance, and the average distortion would be $\mathbb{E}[d(X,\widehat X)]$ where $X$ and $\widehat X$ are independent — usually no better than the $R=0$ bound.

By searching, we condition on the rare event that some particular codeword is jointly typical with $X^n$. Conditioning on a rare event with the right structure pushes the conditional joint towards the desired $P_{X,\widehat X}$. This is the same phenomenon as posterior concentration in Bayesian statistics: a rare event with the right shape effectively reweights the prior into the posterior.

Slide — Binary toy details

The bit strings on the slide are illustrative and were chosen so that one of the four "quick draws" lies relatively far from $X^n$ (Hamming distance 14) and the eventual successful candidate (#14 of 64) lies exactly on the radius-6 ball. In a real run with a fresh random seed the exact distances will differ, but the qualitative behavior is robust:

  • Most random Bernoulli$(1/2)$ codewords lie at Hamming distance close to $n/2$ from any fixed source $X^n$.
  • The fraction within distance $nD$ of $X^n$ is $|B(X^n,nD)|/2^n\approx 2^{-n(1-H_b(D))}$, which is the inverse of $2^{nR(D)}$.
  • So a codebook of size $2^{nR}$ contains a successful match with probability $1-(1-2^{-nR(D)})^{2^{nR}}\approx 1-\exp(-2^{n(R-R(D))})$, which is small if $R<R(D)$ and close to 1 if $R>R(D)$.

For $n=24, D=1/4$: the threshold rate is $R(D)\approx 0.189$ bits/symbol, so we need $2^{0.189\cdot 24}\approx 23$ codewords to have a $\sim 63\%$ chance of finding a match (at the threshold), and $\sim 64$ codewords to be almost sure. The slide's table at 64 codewords is the "almost sure" regime.

Slide — Volume counting threshold

The Hamming-ball volume formula is asymptotic:

\[ |B(0,nD)|=\sum_{k=0}^{nD}\binom{n}{k}\;\le\;2^{nH_b(D)},\qquad D\le 1/2, \]

and the bound is exponentially tight: $|B(0,nD)|\sim 2^{nH_b(D)}/\sqrt{n}$ up to polynomial factors. Plugging into the cover bound gives the threshold rate $R(D)=1-H_b(D)$.

For non-uniform Bernoulli sources (Bernoulli$(p)$ with $p\ne 1/2$), the analogous threshold is $R(D)=H_b(p)-H_b(D)$ for $D\le p$, which reflects the smaller entropy of a biased source.

Looking ahead

Lecture 2 makes the rate–distortion function explicit for two important continuous sources:

  • Gaussian under squared error: $R(D)=\frac{1}{2}\log(\sigma^2/D)$, with the backward Gaussian test channel and the Shannon lower bound (Gaussian is the worst case at fixed variance).
  • Laplacian under absolute error: $R(D)=-\log(\lambda D)$, with the surprising consequence that the optimal reproduction has a Dirac atom at zero — pruning falls out of the rate–distortion theorem.

Both will be derived from the mutual-information formula, using the maximum-entropy property of Gaussians and a clever auxiliary-density trick for Laplace.
