01

Setup and the Rotation Idea

Rotation alone produces a known coordinate distribution

Why Scalar Quantization Fails

Scalar quantization rounds each coordinate independently — assumes uniform spread.

In practice (KV caches, embeddings), a few outlier coordinates dominate energy. Low-bit scalar quantization wastes resolution on outliers.

Random rotation spreads energy: every coordinate gets the same predictable distribution, with typical magnitude $\sim 1/\sqrt{d}$.

Random Rotation Creates a Known Law

Let $\Pi\in\mathbb{R}^{d\times d}$ be Haar-uniform orthogonal. For any fixed $x\in S^{d-1}$:

$$Y \;=\; \Pi x \;\sim\; \mathrm{Unif}(S^{d-1}).$$
Output law depends only on $\|x\| = 1$, not on the direction. The algorithm doesn't need to know the input distribution.

Lemma — Coordinate of a Sphere Point

Lemma 1.

$Y\sim\mathrm{Unif}(S^{d-1})$. Each coordinate $Y_j$ has density

$$f_d(t) \;=\; \dfrac{\Gamma(d/2)}{\sqrt\pi\,\Gamma((d-1)/2)}\,(1-t^2)^{(d-3)/2}, \quad -1\le t\le 1.$$

For large $d$: $\;\sqrt{d}\,Y_j\Rightarrow\mathcal{N}(0,1)$.

Slice argument: surface area at $Y_1=t$ is $\propto(1-t^2)^{(d-3)/2}$. With $t = u/\sqrt{d}$, $(1-t^2)^{(d-3)/2}\approx e^{-u^2/2}$.

Two Distortions

MSE distortion

$$D_{\mathrm{mse}} = \mathbb{E}\|x - \widetilde x\|_2^2.$$

Reconstruction quality. Target $\sim 4^{-b}$.

Inner-product distortion

$$D_{\mathrm{prod}} = \mathbb{E}|\langle y,x\rangle - \langle y,\widetilde x\rangle|^2.$$

Attention, NN search. Want unbiased.

The MSE-optimal quantizer is biased for inner products — we'll fix it in Part 5.

02

MSE TURBOQUANT

Algorithm, anchor, theorem via Panter–Dite

Algorithm: TURBOQUANT$_{\mathrm{mse}}$

Public state: rotation $\Pi$, scalar codebook $\{c_1,\ldots,c_K\}$ for density $f_d$, $K=2^b$.

encode(x): y = Π·x for j = 1..d: idx[j] = argmini |y[j] − ci| return idx # d·b bits decode(idx): for j = 1..d: ỹ[j] = c[idx[j]] return ΠT·ỹ

Codebook offline (Lloyd–Max on $f_d$). No data-dependent training.

One-Bit Anchor: $1 - 2/\pi$

For $Y\sim\mathcal{N}(0,1/d)$ and sign quantizer $\widehat Y = a\,\operatorname{sign}(Y)$:

$$\mathbb{E}[(Y - a\operatorname{sign}Y)^2] = \mathbb{E}Y^2 - 2a\,\mathbb{E}|Y| + a^2, \qquad a^\star = \mathbb{E}|Y| = \sqrt{\tfrac{2}{\pi d}}.$$

At the optimum: $\mathbb{E}[(Y-a^\star\operatorname{sign}Y)^2] = \tfrac{1}{d}(1 - \tfrac{2}{\pi})$.

$$\text{total MSE at } b=1 \;=\; 1 - \tfrac{2}{\pi} \;\approx\; 0.363.$$

Theorem and Proof Outline

Theorem (TURBOQUANT$_{\mathrm{mse}}$).

Every unit $x\in S^{d-1}$:

$$\mathbb{E}\|x - \widetilde x\|^2 \;\le\; \tfrac{\sqrt 3\,\pi}{2}\cdot 4^{-b} \;\approx\; 2.72\cdot 4^{-b}.$$

Step 1. Orthogonal invariance: $\|x-\widetilde x\|^2 = \|Y-\widetilde Y\|^2 = d\cdot C(f_d, b)$, sum of $d$ scalar costs.

Step 2. Panter–Dite high-resolution bound:

$$C(f, b) \;\le\; \tfrac{1}{12}\Bigl(\!\int f^{1/3}\Bigr)^{\!3}\,4^{-b}.$$

Evaluate the integral on $f_d$: $\bigl(\!\int f_d^{1/3}\bigr)^3 = 6\sqrt 3\,\pi/d$, giving the constant $\sqrt 3\pi/2$.

03

Lower Bound

Sphere SLB + Yao $\;\Rightarrow\;$ exponent $4^{-b}$ unbeatable

Sphere SLB

Vector SLB at distortion $D$ and bit budget $B$:

$$D \;\ge\; \dfrac{d}{2\pi e}\cdot 2^{2(h(X) - B)/d}.$$

For $X\sim\mathrm{Unif}(S^{d-1})$, $h(X) = \log_2 A_d$ with surface area $A_d$. Stirling gives $A_d^{2/d}\to 2\pi e/d$, so:

$$D(B) \;\ge\; 2^{-2B/d}.$$

For $B = bd$: $\;D \ge 4^{-b}$ on average over $\mathrm{Unif}(S^{d-1})$.

Yao + Constant Gap

Yao: worst-input expected cost of any randomized algorithm $\ge$ best deterministic algorithm's expected cost on the worst input distribution.

Apply with $\mathrm{Unif}(S^{d-1})$: every randomized $bd$-bit quantizer has some input $x$ with $\mathbb{E}\|x-\widehat x\|^2\ge 4^{-b}$.

$$4^{-b} \;\le\; D_{\mathrm{mse}}^\star(b) \;\le\; D_{\mathrm{mse}}(Q_{\mathrm{mse}}) \;\le\; \tfrac{\sqrt 3\,\pi}{2}\cdot 4^{-b}.$$
Constant gap. Exponent $4^{-b}$ is optimal.
04

Inner-Product Bias

Good MSE doesn't imply good linear-functional preservation

One-Bit MSE Shrinks Projections

Sign quantizer $\widetilde Y_j = a\,\operatorname{sign}(Y_j)$, $a=\sqrt{2/(\pi d)}$. By rotational symmetry $\mathbb{E}[\widetilde x\mid x]=\alpha x$.

$$\alpha = \mathbb{E}\langle x,\widetilde x\rangle = d\,a\,\mathbb{E}|Y_1| = d\cdot\dfrac{2}{\pi d} = \dfrac{2}{\pi}.$$

Therefore $\mathbb{E}\langle y,\widetilde x\rangle = \tfrac{2}{\pi}\langle y,x\rangle$ — biased low.

Geometric Reason — Shrunken Norm

One-bit reconstruction has constant magnitude in every rotated coordinate. Squared norm:

$$\|\widetilde Y\|^2 \;=\; d\,a^2 \;=\; \dfrac{2}{\pi}.$$

So $\|\widetilde x\| = \sqrt{2/\pi} < 1$ — a good representative of its sign cell, but its projection on every direction is multiplicatively shrunk.

Need an unbiased estimator. Enter QJL.
05

QJL and Inner-Product TURBOQUANT

One-bit unbiased sketch, applied to the residual

QJL Definition

Let $S\in\mathbb{R}^{d\times d}$ have iid $\mathcal{N}(0,1)$ entries. For $u\in S^{d-1}$:

$$Q_{\mathrm{qjl}}(u) \;=\; \operatorname{sign}(Su) \;\in\; \{-1,+1\}^d, \qquad Q_{\mathrm{qjl}}^{-1}(z) \;=\; \dfrac{\sqrt{\pi/2}}{d}\,S^\top z.$$

One bit per coordinate. Unbiased estimator of $u$.

Lemma — Unbiased with $1/d$ Variance

Lemma (QJL).

$$\mathbb{E}_S\langle y,Q_{\mathrm{qjl}}^{-1}(Q_{\mathrm{qjl}}(u))\rangle = \langle y,u\rangle, \qquad \operatorname{Var}_S(\cdot)\;\le\;\dfrac{\pi}{2d}\|y\|^2.$$

Unbiasedness. Gaussian identity $\mathbb{E}\bigl[s\operatorname{sign}(s^\top u)\bigr] = \sqrt{2/\pi}\,u$ for $s\sim\mathcal{N}(0,I)$.

Variance. Each summand $\le\tfrac{\pi}{2}\|y\|^2$ in second moment; $d$-fold averaging gives $\tfrac{\pi}{2d}\|y\|^2$.

Two-Stage Decomposition

Write $x = \widetilde x_{\mathrm{mse}} + r$ where $\widetilde x_{\mathrm{mse}}$ is MSE-quantized:

$$\langle y, x\rangle \;=\; \langle y, \widetilde x_{\mathrm{mse}}\rangle \;+\; \langle y, r\rangle.$$

First term: from the MSE-quantized vector. Second: estimate by QJL on the residual $r$.

QJL variance scales with $\|r\|^2$. The MSE stage makes $\|r\|$ small, so QJL is accurate precisely when MSE worked.

Algorithm and Theorem

encode(x, b): idx = TQ_mse(x, b−1) # (b−1)·d bits r = x − TQ_mse_inv(idx); γ = ‖r‖ q = sign(S·r) in {−1,+1}^d # d bits return (idx, q, γ) decode(idx, q, γ): return TQ_mse_inv(idx) + γ·(√(π/2)/d)·ST·q

Theorem (TURBOQUANT$_{\mathrm{prod}}$).

$$\mathbb{E}\langle y,\widetilde x\rangle = \langle y,x\rangle,\qquad D_{\mathrm{prod}} \;\le\; \dfrac{\sqrt 3\,\pi^2\,\|y\|^2}{d}\cdot 4^{-b}.$$

Inner-product errors decay $d$ times faster than MSE errors.

06

KV Cache Quantization

Online, data-oblivious, attention-preserving

Why TURBOQUANT Fits the KV Cache

Decoder-only transformer: each new token uses inner products $\langle q_t, k_s\rangle$ between query and past keys.

Attention preservation $\Leftrightarrow$ inner-product preservation.

  • Online: each new key quantized as it arrives.
  • No calibration: codebook universal, not dataset-specific.
  • Inner-product control: unbiased, $\mathcal{O}(1/d)$ variance.
Quality-neutral around $3.5$ bits/channel; gradual degradation near $2.5$.
Daliri et al., "TurboQuant: Online Vector Quantization with Optimal Distortion", arXiv:2504.19874 (2025).

Recap

  • Rotation: turns any unit vector into a sphere point with known coordinate density $f_d$.
  • MSE TURBOQUANT: Lloyd–Max on $f_d$ achieves $D_{\mathrm{mse}}\le\tfrac{\sqrt 3\pi}{2}\cdot 4^{-b}$.
  • Lower bound: sphere SLB + Yao $\Rightarrow$ exponent $4^{-b}$ optimal.
  • Bias: MSE shrinks projections by $2/\pi$ at one bit.
  • QJL: one-bit unbiased linear-functional estimator with $\mathcal{O}(1/d)$ variance.
  • IP TURBOQUANT: two-stage; unbiased, $D_{\mathrm{prod}} \le \tfrac{\sqrt 3\pi^2\|y\|^2}{d}\cdot 4^{-b}$.
