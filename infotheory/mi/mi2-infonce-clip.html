01

InfoNCE

Noise-contrastive bound on mutual information

Setup — One Positive, $K-1$ Negatives

Sample $(x,y)\sim p_{X,Y}$ and $K-1$ negatives $\{y_k^-\}\stackrel{\mathrm{iid}}{\sim}p_Y$, independent of $x$.

Critic $f(x,y)$. Loss:

$$\mathcal{L}_{\mathrm{InfoNCE}}(f) \;=\; -\mathbb{E}\log\dfrac{e^{f(x,y)}}{e^{f(x,y)} + \sum_{k=1}^{K-1}e^{f(x,y_k^-)}}.$$
A $K$-class softmax cross-entropy: classify the true $y$ among $K$ candidates.

InfoNCE Bound

Theorem (van den Oord et al., 2018).

For any critic $f$:

$$I(X;Y) \;\ge\; \log K - \mathcal{L}_{\mathrm{InfoNCE}}(f).$$

Since $\mathcal{L}_{\mathrm{InfoNCE}}\ge 0$, the bound is $\le \log K$.

To estimate $I=10$ nats: need $K\ge e^{10}\approx 22{,}000$ negatives. CLIP uses $K=32{,}768$.

van den Oord, Li, Vinyals, "Representation Learning with Contrastive Predictive Coding", arXiv 2018.

Proof Sketch

Optimal critic: $f^\star(x,y) = \log r(x,y) + c(x)$. Substitute:

$$\mathcal{L}_{\mathrm{InfoNCE}}(f^\star) \;=\; -\mathbb{E}\log\dfrac{r(x,y)}{r(x,y) + \sum_{k=1}^{K-1}r(x,y_k^-)}.$$

Note $\mathbb{E}_{p(y_k^-)}[r(x,y_k^-)] = 1$. Apply Jensen on the convex $-\log$:

$$\mathcal{L}_{\mathrm{InfoNCE}}(f^\star) \;\ge\; \log K - I(X;Y) + o(1).$$

Rearranging yields $I(X;Y) \ge \log K - \mathcal{L}_{\mathrm{InfoNCE}}(f^\star)$. The bound holds for any $f$ since $f^\star$ minimizes the loss.

Optimal Critic = Log-Density-Ratio

The infimum is attained at any $f^\star$ with

$$f^\star(x,y) \;=\; \log r(x,y) + c(x) \;=\; \log\dfrac{p(y\mid x)}{p(y)} + c(x).$$
  • Same log-density-ratio as DV's optimal critic.
  • $\mathrm{softmax}(f^\star)$ approximates the conditional $p(y\mid x)$.
  • Bound is biased low by at least $\max(0, I-\log K)$ at high MI.

CLIP Architecture

InfoNCE deployed at web scale

Two Encoders, One Embedding Space

Trained jointly on 400M image–text pairs:

Image $x$
→
$g_\mu$
→
$g_\mu(x)\in\mathbb{R}^d$
Text $y$
→
$h_\phi$
→
$h_\phi(y)\in\mathbb{R}^d$
Radford et al., "Learning Transferable Visual Models From Natural Language Supervision", ICML 2021.

Separable Critic and Symmetric Loss

Cosine similarity, scaled by learned temperature $\tau$:

$$f_{\mu,\phi}(x,y) \;=\; \tfrac{1}{\tau}\,g_\mu(x)^\top h_\phi(y),\qquad \|g_\mu(x)\| = \|h_\phi(y)\| = 1.$$

Minibatch of $N$ pairs. Similarity matrix $S_{ij} = f(x_i, y_j)$. Symmetric loss:

$$\mathcal{L}_{\mathrm{CLIP}} \;=\; \tfrac{1}{2}\bigl(\mathcal{L}_{x\to y} + \mathcal{L}_{y\to x}\bigr).$$

Each direction is one InfoNCE pass over the batch.

$N\times N$ Similarity Matrix

Diagonal: true pairs, should rank highest in their row and column.

Off-diagonal: contrastive negatives — random shufflings within the batch.

$N$ row + $N$ column classifications, optimized simultaneously by one matrix multiply.

03

Zero-Shot Classification

Reusing the contrastive head, no task-specific training

CLIP as a Classifier

$K$-way classification with labels $\{c_1,\ldots,c_K\}$:

  1. Build prompts: $y_k = $ "A photo of a $\{c_k\}$.".
  2. Embed prompts: $h_k = h_\phi(y_k)$.
  3. Embed test image: $g = g_\mu(x)$.
  4. Predict $\hat c = \arg\max_k\; g^\top h_k$.

No retraining. Classifier built from text alone.

Why It Works — InfoNCE Approximates MAP

InfoNCE training drives $g^\top h \approx \log r(x,y) + c(x)$. For a class label:

$$g_\mu(x)^\top h_\phi(y_k) \;\approx\; \log\dfrac{p(y_k\mid x)}{p(y_k)} + \text{const}.$$

$\arg\max_k$ approximates the MAP class, modulo the marginal $p(y_k)$.

Near-optimal Bayes classifier when label priors are roughly uniform across prompts.

Prompt Engineering Matters

Prompt styleImageNet top-1
Single word ("dog")~60%
"A photo of a {label}."~68%
Ensemble of 80 templates~76%

Comparison: ResNet-50 supervised on ImageNet $\approx$ 76% top-1.

Practice

Bias, variance, batch size

Four Bounds Side by Side

BoundTrade
BATight if $q$ is good; needs $H(X)$ for MI numbers
DVTight; variance $\sim e^I/N$ (explodes at high MI)
NWJLooser than DV; variance polynomial in $I$
InfoNCECeiling $\log K$; very stable variance

InfoNCE wins on training stability; DV and NWJ on tight numerical estimation.

Why Batch Size Matters

Bound caps at $\log K$. To recover $I$ nats, need $K\ge e^I$.

  • SimCLR: $K\sim 4{,}096$, accelerator memory.
  • MoCo: queue of $K\sim 65{,}000$ past negatives, decoupled from batch.
  • CLIP: $K = 32{,}768$ at full scale.
Tightness gain saturates once $\log K\gg I$. Compute cost grows $\mathcal{O}(K)$.

Recap

  • InfoNCE: $K$-way softmax. Bound $I\ge \log K - \mathcal{L}$.
  • Optimal critic: $f^\star = \log r + c$, same as DV.
  • $\log K$ ceiling: biased low at high MI; variance very stable.
  • CLIP: separable critic, symmetric loss, $N\times N$ similarity matrix.
  • Zero-shot: text encoder defines classifier; matches supervised ResNet-50.
  • Practice: InfoNCE for training; NWJ for tight estimation; BA for likelihoods.
